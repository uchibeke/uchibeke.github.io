UTranslate App


Problem:

What happens if someone gets access to your phone and opens your text? How about emails you prefer only the receiver the decipher? UTranslate helps to covert readable english texts to a specified "language" which can only be deciphered if you have the key. It is:

  1. A hybrid App allowing users to map English alphabets to one or more letters/numbers/symbols of any existing language supported on the user’s device.
  2. Translates user input to the defined ‘language’ in real time.
  3. Inspired by my high school experience learning “jungle language” in Nigeria.
  4. Initiated, Designed and implemented.

Strategy and goals:

Designed an algorithm that takes user input (string), converts it to a dynamic array putting each character at each index of the array and then converting to a specified langauge. This is just an abstraction of what happens. There is more to it. However, the algorithm is efficient and translation happens instantly as your type.
