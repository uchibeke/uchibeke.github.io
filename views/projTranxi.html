TRANXI

Tranxi is inspired by Jungle Language, a language we often resorted to as high school kids in Port harcourt, Nigeria, were I was born. We did this to conceal what we are talking about.
In the most basic sense of Jungle language, the vowels are mapped to numbers: a = 1, e = 2, i = 3, o = 4, and u = 5. Each of the consonants, on the other hand, remain as is, however, an 'a' is appended to each of them: B becomes Ba, C becomes Ca, D becomes Da, F becomes Fa ... and Z becomes Za.

MY ROLE

Tranxi is an independent project. I worked on all the design and implementation. Because the inspiration for Tranxi was from personal experience, I drew knowledge from that experience in the implemenation of Tranxi.

Experience Strategy and Vision

I designed an algorithm that takes user input (string), inserts it to a dynamic keyed dictionary, compares each element to a second dictonary using the key and then converts to a specified langauge. This is just an abstraction of what happens. There is more to it. However, the algorithm is efficient and translation happens instantly as the user types.

Planning

Design and Execution

Testing

THE CHALLENGE

Provide a hybrid App that allows users to map English alphabets to one or more letters/numbers/symbols of any existing language supported on the user’s device.

App should be able to translate user input to the defined ‘language’ in real time.

Jungle Language Cheat Sheet

In Jungle Language, if I wanted to write
How are You?,
I would say
Ha4Wa 1Ra2 Ya45?
The vowels A, E, I, O, U in jungle language are represented by 1, 2, 3, 4, 5 respectively. The consonants, on the other hand, remain as is, however, an 'a' is appended to them - B becomes Ba, C becomes Ca, D becomes Da, F becomes Fa ... and Z becomes Za.
The goal of Tranxi is to allow people to create their own language just like someone came up with Jungle Language.
